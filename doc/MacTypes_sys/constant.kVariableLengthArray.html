pub const kVariableLengthArray: ItemCount

Array bounds: variable length array

Note

kVariableLengthArray is used in array bounds to specify a variable length array. It is ususally used in variable length structs when the last field is an array of any size. Before ANSI C, we used zero as the bounds of variable length array, but zero length array are illegal in ANSI C. Example usage:

struct FooList { short listLength; Foo elements[kVariableLengthArray]; };
